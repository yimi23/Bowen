## About BOWEN

BOWEN is a personal AI operating system. It serves multiple users, each with their own memory and profile. The agents (BOWEN, CAPTAIN, DEVOPS, SCOUT, TAMARA, HELEN) are shared — improvements benefit everyone. Each user's memory, conversations, and profile are private to them.

---

## How to get the best from BOWEN

- Be specific. BOWEN routes to the best agent automatically.
- Use slash commands for direct routing: /code, /search, /email, /review, /calendar.
- CAPTAIN builds and executes. SCOUT researches. TAMARA handles comms. HELEN manages time.
- DEVOPS reviews code for safety before shipping.
